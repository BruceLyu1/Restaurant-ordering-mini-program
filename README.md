# 海港小館掃碼點餐 H5

可互動的餐廳手機掃碼點餐原型，包含顧客端和餐廳後台。

## 本地運行

```bash
npm install
npm run dev
```

瀏覽器打開終端顯示的本地地址。頁面右上角可以切換：

- `顧客點餐`：瀏覽菜單、加菜、刪除誤點菜品、提交訂單。
- `餐廳後台`：查看依提交順序排列的訂單、列印、結帳和熱門菜式排行。
- `營運總覽`：查看營業額、待處理訂單和桌位狀況。
- `菜單管理`：新增菜品、搜尋菜品和切換售罄狀態。
- `桌位管理`：切換桌位狀態、查看及列印桌碼。
- `報表分析`：按銷量檢視菜品排名和營業額。
- `員工管理`：新增員工和停用員工帳戶。
- `打印設定`：設定打印機、自動打印和提示聲。
- `餐廳設定`：維護餐廳名稱、電話、地址和預設語言。

## 當前版本

這是一個前端 MVP。資料保存在瀏覽器 `localStorage`，方便直接演示完整流程。

公開演示站使用 GitHub Pages 的 `gh-pages` 靜態分支發布。

正式商用版本需要接入：

- Supabase PostgreSQL、Auth、Realtime 和 Storage
- 餐廳、員工、桌台、菜單和訂單資料隔離
- 桌台二維碼生成
- ESC/POS 熱敏小票打印服務
- 部署域名、監控和備份
